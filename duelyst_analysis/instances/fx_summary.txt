Key Metadata Extracted:
----------------------
- rsx_name: The RSX identifier (e.g., fx_f1_holyimmolation)
- fx_type: Type of effect (sprite_animation, particle, etc.)
- category: Usage category (faction-specific, buff, damage, etc.)
- file_path: Path to the sprite sheet image
- frame_prefix: Prefix for animation frames in plist
- frame_delay: Delay between frames (duration per frame)
- is_16bit: Whether the texture uses 16-bit color
- has_plist: Whether there's a plist file with sprite data
- related_sounds: Related sound effect resources
- usage_context: Which cards use this FX
- used_by_cards: Count of cards using this FX

Common FX Patterns:
------------------
1. Bloodborn Spell FX (BBS): fx_f{faction}_bbs_{ability}
2. Faction Spell FX: fx_f{faction}_{spellname}
3. Generic Effects: fxImpact*, fxExplosion*, fxTeleport*
4. Tile/Ground Effects: fxBloodGround, fxShadowCreepSpawn
5. Buff Effects: ForceField, FloatingShield, fxBuffSimpleGold

Notes:
------
- Frame delays are typically 0.08 seconds (80ms), some are faster (0.04) or slower (0.1-1.0)
- Most FX are 16-bit textures for better quality
- All FX use plist files for sprite sheet definitions
- Generic FX like impacts and explosions are reused across multiple cards
