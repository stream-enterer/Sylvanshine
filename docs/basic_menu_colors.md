# Basic Menu Colors

## Menu Window Background (Dialog Body)

| Property | Value |
|----------|-------|
| Color | `0x00002f7f` |
| RGB | (0, 0, 47) - deep blue |
| Opacity | `0x7f` = 127/255 = ~50% |

## Title Bar Gradient

3-color vertical gradient from top to bottom:

| Position | Hex | RGB | Opacity |
|----------|-----|-----|---------|
| Top | `0x0060bf7f` | (0, 96, 191) - cyan/blue | ~50% |
| Middle | `0x0000507f` | (0, 0, 80) - dark blue | ~50% |
| Bottom | `0x00f0ff7f` | (0, 240, 255) - bright cyan | ~50% |

The gradient is darkest in the middle and brighter at the edges.
